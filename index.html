Array Manipulation

Array can defined as a container that store the same data types. data type can be an int, float, string, char and so on. Within an array it is not allowed to store different data types. before trying to store data, we need to reserve a memory which is equvalent to the data going to stored. If the memory reserved is less than the required amout of memory, then some of the data could be discarder, as a result, the information distoreted that would let us to commite mistake in our analysis. Whereas, if we reserved more space than required, it is also waste of memory. Our system try to use the required amout of memory and the remaining space remain free but the operating system must wait untill the process finished and free the memory.In between, the free memory is a waste.

Introduction to Loops

As discussed above, Once the memory is reserved for the data, then either we need to store or retrive the data form the array. Inorder to do such array manipulation we need to use loops. There are different types of loops with their specific functionalities.

To list some of them:

  • For loop:- the syntax for such type of loop is
    for (statement_1; statement_2; <>statement_3) {
    task to be performed;
    }
  • While loop:- the syntax for such type of loop is
    while(condition meet) {
    performed task;
    }
  • do_while loop:- the syntax for such type of loop is
    do{ performed task }while (condition meet);

For Loop

In a for loop we must first declare the initial condition to start the loop for example, int i=0; in this expresion, variable i with int data type is decleared and assigned its initial value to 0. the second parameter for the for loop is the loop control part. As discribed above, for example int n=20; and for the loop controling ,if we set it as {i less than n}; then the loop iterates and perform the task which is defined inside the brace as far as the condition is satisfied.

The general syntax for such loop type is as follow:

for (statement_1; statement_2; statement_3)
{
task to be performed;
}

Where:

  • statement_1:-is excuted one time before the program get in to the task to be performed.
  • statment_2:-it defines the condition excute the task to be performed.
  • statment_3:-excuted every time after the task excuted.

While Loop

For a while loop, while the condition satisfied the program inter in to the task. For every iteration the program checks the condition then once the condition meets it perform the task and check again the condition for next excution. The loop termination terminated when the program does not able to satisfiy the while loop condition set by the programmer.

Do While Loop

The main difference between the while loop and do while loop,for the later one the program excute once without checking the condition. then after it tries to check the condition set by the programmer.If it satisfiy the condtion then the program goes to the next task excution but if it fails to satisfiy the condition, it will automatically stop the excution.
